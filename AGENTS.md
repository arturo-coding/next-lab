# Next.js: ALWAYS read docs before coding

Before any Next.js work, find and read the relevant doc in `node_modules/next/dist/docs/`.
Your training data is outdated — the docs are the source of truth.

## Documentation Structure
The bundled docs mirror the structure of the Next.js documentation site:
- `01-app/`: App Router guides and API reference
- `02-pages/`: Pages Router guides
- `03-architecture/`: Internal architecture and design principles
- `index.md`: Overview and entry point
